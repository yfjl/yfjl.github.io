electron快速入门

发表于 2017-11-09 | | 阅读次数:

Electron 是一款可以利用 Web 技术 开发跨平台桌面应用的框架，最初是 Github 发布的 Atom 编辑器衍生出的 Atom Shell，后更名为 Electron 。
Electron 提供了一个能通过 JavaScript 和 HTML 创建桌面应用的平台，同时集成 Node 来授予网页访问底层系统的权限。目前常见的有 NW、heX、Electron，可以打造桌面应用。

扩展

electron除了使用前端技术实现界面展示的功能外，还提供了大量的桌面环境接口。比如支持Notification，Jump List, dock menu等。具体支持哪些桌面接口以及如何使用 参考中文教程：https://electron.org.cn/doc/index.html

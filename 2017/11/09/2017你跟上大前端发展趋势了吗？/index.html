2017你跟上大前端发展趋势了吗？

发表于 2017-11-09 | | 阅读次数:

2016一年都在忙于搞node.js，也算是边学边用吧。2017上半年学了java，spring等，用java也搞了个商城。貌似现在已经忘记java了….脑子是个好东西，可惜就是记不住….
前端新技术层出不穷，都在2017补补吧！所谓学新技术，无非就是学学它们的api，会使用、仅此而已。
这样的学习方式，何时能成为大神呢？永远的码农，我想没有人愿意。研究框架的源码，精通js基础，熟练掌握23种设计模式，学学数据结构与算法….

哦！！！貌似工作3年多的我，还是菜鸟。

1、微信小程序

微信小程序是一种可以媲美原生app的“app”，上市当天便在业内引起较大轰动。其实早在小程序之前也有百度轻应用这样的产品，很遗憾并未做起来。
究竟小程序会爆发多大的威力，我们拭目以待。但是可以肯定的是，一定会催生大量的小程序开发人员。而小程序用到的开发语言，就是我们前端的Javascript。相对于其他开发人员，这也是前端开发的一波红利。

2、Vue.js/react.js

Vue.js作为国产框架，推荐使用

3、ES6/Typescript

ES6是JavaScript语言的下一代标准。Typescript是js的超集。

4、支付宝小程序

支付宝小程序和微信小程序语法很像。
教程：https://docs.alipay.com/mini/developer/getting-started
demo： https://github.com/yfjl/myMovie

5、Electron

Electron 提供了一个实时构建桌面应用的纯 JavaScript 环境。简单来说，你可以使用Electron 来写一个桌面程序。比如：一个可以运行在windows上的百度云软件。
github上的star 将近53k，可谓异常火爆啊。
高兴的是，现在electron支持window 7和之上版本,osx 10.9之后,Linux等等，各个平台大部分几乎都做到了支持。
中文教程地址：https://electron.org.cn/doc/index.html

6、weex

Weex 是一套简单易用的跨平台开发方案，能以 web 的开发体验构建高性能、可扩展的 native 应用，为了做到这些，Weex 与 Vue 合作，使用 Vue 作为上层框架，并遵循 W3C 标准实现了统一的 JSEngine 和 DOM API，这样一来，你甚至可以使用其他框架驱动 Weex，打造三端一致的 native 应用。
教程：http://weex.apache.org/cn/guide/

7、egret/cocos2d-js

作为web前端，我们有一颗作为全栈的心。有谁仅仅只满足当前的技术呢？推荐学习egret/cocos2d-js 游戏引擎。我个人学过egret。
egret使用typescript语法编写游戏，可以搞2d和3d游戏。看起来很强大的样子。cocos2d-js  暂时还没用过，不过吗？貌似很火。
教程：百度搜索关键字吧。
egret Demo:  https://github.com/yfjl/egret-aircraft

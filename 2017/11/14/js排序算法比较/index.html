js排序算法比较

发表于 2017-11-14 | | 阅读次数:

前端也要学学算法啊！算法是程序的核心，虽然工作中我们使用算法的地方不多。每天都学新知识也无非学学新的api。想成为一个正真的程序员，一个更牛的程序员，我觉得还是要学学算法。学起来枯燥，但却充满乐趣。

在javascript数据结构与算法这本书中，提供了5中排序算法。分别是：冒泡排序、选择排序、插入排序、归并排序和快速排序。
但在算法导论中，又多了线性和时间排序。本文将详解各种排序算法，且持续更新….
对于各种排序算法性能问题，后面将会进行测试。

1、冒泡排序

比较任何两个相邻的项，如果第一个比第二个大，则交换它们

改进后的冒泡排序

如果从内循环减去外循环中已跑过的轮数，就可以避免内循环中所有不必要的比较

2、选择排序

思路：找到数据结构中的最小值并将其放置在第一位，接着找到第二小的值并将其放在第二位，以此类推。

3、插入排序

假定第1项已经排序了,和第0项比较，如果第0项大于第一项，那么第1项与第0项交换位置。此时第1项数大一些。第2项和第1项进行比较，如果第1项大于第2项，第2项与第1项交换位置。换位后通过while循环，此时如果
第1项仍然小于第0项，那么交互位置。依次类推，最小的数在最前面。

4、归并排序

归并排序是一种分治算法，其思想是将原始数组切分成较小的数组，直到每个小数组只有一个位置，接着将小数组归并成较大的数组，直到最后只有一个排序完毕的大数组。

结果：

由图可以清晰的看到，首先把数据分散成最小单元，然后不断的合并排序。

5、快速排序

快速排序3步：
1、在需要排序的数组中，任选一个元素作为“基准”
2、将小于“基准”和大于“基准”的元素分别放到两个新的数组中，等于“基准”的元素可以放在任一数组
3、对于两个新的数组不断重复第一步第二步，直到数组只剩下一个元素，这时step2的两个数组已经有序，排序结果也很容易得到了（leftArray+基准元素+rightArray）

以数组[1, 2, 5, 4, 3]举例，第一次排序，找个基准，基准可以是数组的任意元素，为了方便说明，可以选择第一个元素，这里我以中间元素举例。于是找到5为基准，小于5和大于5的分别放到两个新的数组中，等于5的可以放到任意一边 。

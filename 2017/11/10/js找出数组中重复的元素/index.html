js找出数组中重复的元素

发表于 2017-11-10 | | 阅读次数:

题目描述：

找出数组arr中重复出现过的元素

function duplicates(arr) {

}
输入：
[1, 2, 4, 4, 3, 3, 1, 5, 3]
输出：
[1, 3, 4]

面试题可能会出现。

解析：

实现方式当然有很多种，现在讲解一种。

结果：[1,4,3]

思路：

indexOf() 方法可返回某个指定的字符串值在字符串中首次出现的位置。
stringObject.indexOf(searchvalue,fromindex)
searchvalue：必须，要检测字符串的值
fromindex：可选，字符串开始检索位置

lastIndexOf() : 从尾部开始检索

循环变量数组没有问题，主要是 arr.indexOf(elem) !=arr.lastIndexOf(elem) 这句话。
假如 2 在数组中只有一个，那么无论是indexOf 还是lastIndexOf,其返回的位置都是相同的。那么就可以说明，该元素在数组中确实不重复。
相反，则不相同，说明有相同的元素。
那么 result.indexOf(elem) == -1 是什么意思呢？
indexOf = -1 说明这个元素在数组中不存在，检索不到。那么如果存在就不会返回-1。所以起到一个去重的效果。
